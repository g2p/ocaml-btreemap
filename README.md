# ocaml-btreemap

An [ocaml-rs](https://github.com/zshipko/ocaml-rs) wrapper around the Rust `BTreeMap` type for use in OCaml

## Building

    make

to run the tests:

    make test

or to build manually:

    cargo build
    jbuilder build
    jbuilder runtest

## Installation

    opam pin add btreemap .


## API

See `src/btreemap.mli`
